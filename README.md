# Mini Project - Sidebar Menu (HTML & CSS)

## Overview
This project is a responsive sidebar navigation menu built using HTML and CSS. The sidebar can be opened and closed using a menu button and includes smooth animations, hover effects, and social media icons.

## Features
- Responsive Sidebar Menu
- Smooth Open/Close Animation
- Font Awesome Icons
- Hover Effects
- Modern UI Design
- Pure HTML & CSS (No JavaScript)

## Technologies Used
- HTML5
- CSS3
- Font Awesome
- Google Fonts (Poppins)

## Project Structure
 index.html  - Main HTML file
style.css   - Styling file
photo.jpg   - Background image
